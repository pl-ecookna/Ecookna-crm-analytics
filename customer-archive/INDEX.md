# Архив документации

Эта папка содержит копию актуальной документации проекта для передачи заказчику.

## Что включено

- `README.md`
- `apps/api/README.md`
- `apps/api/.env.example`
- `apps/api/docs/*`
- `apps/api/tests/fixtures/README.md`
- `apps/web/README.md`
- `apps/web/.env.example`
- `docs/INDEX.md`
- `docs/customer-handover.md`
- `docs/infrastructure/dokploy.md`
- `docs/monorepo/*`
- `.env.example`

## Что намеренно не включено

- `docs/private/*`
- файлы с секретами и учетными данными
- исходный код приложения
- тяжелые артефакты вроде `node_modules` и lockfile
